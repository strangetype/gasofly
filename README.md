# Gasofly

TypeScript проект на Vite с настроенным Prettier.

## Установка

```bash
pnpm install
```

## Разработка

Запуск dev-сервера:

```bash
pnpm dev
```

Сервер запустится на `http://localhost:3000`

## Сборка

```bash
pnpm build
```

## Предпросмотр сборки

```bash
pnpm preview
```

## Форматирование кода

```bash
pnpm format
```

## Структура проекта

```
gasofly/
├── src/              # Исходный код
│   ├── main.ts       # Входная точка приложения
│   └── style.css     # Стили
├── index.html        # HTML шаблон
├── vite.config.ts    # Конфигурация Vite
├── tsconfig.json     # Конфигурация TypeScript
├── .prettierrc       # Конфигурация Prettier
└── package.json      # Зависимости проекта
```

## Технологии

-	TypeScript
-	Vite
-	Prettier (табы по 4 пробела)
